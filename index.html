Tentang saya

Perkenalan saya

Saya adalah lulusan Teknik Informatika dari Universitas Trunojoyo Madura dengan keahlian di bidang pemrograman bahasa pemrograman : PHP, HTML/CSS, JavaScript, dan jQuery. Saya berpengalaman dalam menggunakan Laravel dan CodeIgniter untuk mengembangkan aplikasi yang kuat dan terukur. Selain itu, saya memiliki keterampilan dalam manajemen basis data dengan MySQL dan PostgreSQL, serta analisis data menggunakan Tableau dan Power BI.

Selama magang di SDN Lombang Dajah 1 Blega, saya mengembangkan sistem informasi sekolah yang komprehensif menggunakan PHP dan Laravel, yang secara signifikan meningkatkan efisiensi pengelolaan data sekolah, termasuk data data siswa, guru, kegiatan ekstrakurikuler, dan penghargaan. Saya juga mengintegrasikan visualisasi data menggunakan Tableau ke dalam sistem.

Selain itu, saya memiliki pengalaman dalam desain UI/UX menggunakan Figma, yang saya pelajari dalam mata kuliah Interaksi Manusia dan Komputer selama masa studi saya.
